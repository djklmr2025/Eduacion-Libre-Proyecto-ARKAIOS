🔑 API, Bridge Keys y Documentación

Este módulo puede ser llamado desde EduAgent, ARKAIOS Core, o cualquier sistema externo usando los siguientes métodos:

1. URL con parámetros (más simple)
https://eduacion-libre-proyecto-arkaios.vercel.app/generador-ia-imagenes.html ?prompt=beautiful+fantasy+warrior,+epic &resolution=768x768 &count=4 &nsfw=0 &autostart=1
2. PostMessage Bridge (desde otro iframe/ventana)
// Enviar petición: window.postMessage({ type: 'ARKAIOS_IMG_REQUEST', token: 'TU_BRIDGE_TOKEN', // del panel izquierdo prompt: 'dragons flying over mountains', style: 'fantasy art, epic, 8k', resolution: '768x768', count: 4, nsfw: false }, '*'); // Recibir imágenes generadas: window.addEventListener('message', (e) => { if (e.data.type === 'ARKAIOS_IMG_RESPONSE') { const urls = e.data.images; // array de URLs temporales console.log('Imágenes:', urls); } });
3. API Perchance directa (desde servidor)
// Paso 1: Obtener userKey (tu canal personalizado) GET https://image-generation.perchance.org/api/verifyUser ?channel=xnaxg7ceyj &__cacheBust=1234567890 // Paso 2: Generar imagen POST https://image-generation.perchance.org/api/generate Content-Type: application/x-www-form-urlencoded prompt=beautiful+warrior&negativePrompt=blurry& resolution=768x768&guidanceScale=7&seed=42& channel=xnaxg7ceyj&userKey=KEY_DEL_PASO1& adultContent=0 // 1 = NSFW habilitado // Paso 3: Descargar imagen GET https://image-generation.perchance.org/api/downloadTemporaryImage ?imageId=ID_DEL_RESPONSE
4. Integración con EduAgent (api/edu-agent.js)
// POST /api/edu-agent con mode:'ai_generate' // activa Perchance en vez de Pexels: { "request": "9 imágenes de volcanes para 6to grado", "mode": "ai_generate", "bridge_token": "TU_TOKEN" }

⚠️ Las URLs de imágenes generadas son temporales (~5 min). Para persistirlas descárgalas o súbelas a tu propio storage.
